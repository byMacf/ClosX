# CLOSFabricGenerator

## Command Line Operation
```
  5 Tier CLOS
  $ python3 closgen.py -t 5 -s 4 -t4 8 -t3 12 -t2 16 -t1 20

  3 Tier CLOS
  $ python3 closgen.py -t 3 -s 8 -t2 16 -t1 16

    -t: number of tiers
    -s: size of spine
    -t4: size of tier 4
    -t3: size of tier 3
    -t2: size of tier 2
    -t1: size of tier 1
```
